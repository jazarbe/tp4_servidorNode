Hola, estas son las instrucciones para correr nuestro proyecto.
1. Abrir el código
2. En la consola, escribir "npm i"
3. En la consola, escribir "npm run start"
4. Va a aparecer un link ("http://localhost:3000"), hacer ctrl + click (o abrirlo en un navegador)
5. Desde la barra de búsqueda, agregar luego del link base "/hora" para ver la hora actual
6. Desde la barra de búsqueda, agregar luego del link base "/fecha-completa" para ver la fecha y hora actuales
7. EXTRA: Desde la barra de búsqueda, agregar luego del link base "/cualquiercosa" (cualquier cosa que quieran poner distinto de las opciones anteriores), para una sorpresa!
